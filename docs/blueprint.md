# **App Name**: Charak Srivastava Portfolio

## Core Features:

- Interactive Resume Display: Displays resume information in an engaging and interactive format, pulled from the provided resume image.
- Project Portfolio: Showcase personal projects with descriptions and links, including the portfolio itself.
- AI-Powered Recommendation Tool for Project Suggestions: Generates personalized project ideas based on the user's skillset and interests, suggesting projects to further develop skills.
- Stats Section: Display key metrics and achievements, such as projects completed, skills proficiency, etc.
- About Me Section: Introduce Charak Srivastava with a brief summary of skills and experiences.
- Contact Section: Provide contact information and links to social media profiles like LinkedIn, GitHub, and Instagram.

## Style Guidelines:

- Primary color: Dark Blue (#2E3A87), conveying professionalism and competence.
- Background color: Light Gray (#EAEAEA), providing a clean and modern backdrop.
- Accent color: Teal (#309386), used for highlights and interactive elements to draw attention.
- Headline font: 'Space Grotesk' (sans-serif) for a modern and technical feel. Body font: 'Inter' (sans-serif) to ensure readability.
- Use modern and minimalistic icons for navigation and social media links.
- A clean, single-page layout with clear sections for easy navigation and readability.
- Subtle animations for transitions and interactions to enhance user engagement without being distracting.
